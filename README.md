# fintech

AI in FinTech credit scorecard presentation dashboard (RiskLens).

## Run locally

```bash
cd fintech-dashboard
npm install
npm run dev
```

## Deploy

Build: `npm run build` — output in `fintech-dashboard/dist/`.

Deploy `fintech-dashboard` on [Vercel](https://vercel.com) or [Netlify](https://netlify.com) with root directory `fintech-dashboard`, build command `npm run build`, publish directory `dist`.
